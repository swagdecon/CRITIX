CRITIX

CRITIX is a web application built with React and Java where users can browse through a vast collection of movies and TV shows. The app also allows users to add movies to their watchlist, view movie details, and rate and review movies they have watched.

Features

View movie details, including the synopsis, release year, runtime, and user reviews
Search for movies and TV shows by title, actor, or director
View a list of the most popular movies and TV shows at the moment
Get recommendations based on currently selected movies
Technologies Used
Frontend: React
Backend: Java (Spring Boot)
Database: MongoDB
Visit http://www.critix.app to view the app.
