Popflix

Popflix is a web application built with React and Java where users can browse through a vast collection of movies and TV shows. The app also allows users to add movies to their watchlist, view movie details, and rate and review movies they have watched.

Features

View movie details, including the synopsis, release year, runtime, and user reviews
Search for movies and TV shows by title, actor, or director
View a list of the most popular movies and TV shows at the moment
Get recommendations based on currently selected movies
Technologies Used
React for the front-end
Java for the back-end
MongoDB for the database
Getting Started
Clone the repository from Github to your local machine.
Install dependencies with npm install in the client directory and mvn install in the server directory.
Start the development server with npm start in the client directory and mvn spring-boot:run in the server directory.
Visit http://localhost:3000/ in your browser to view the app.
